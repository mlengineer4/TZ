## Что такое цикл

**Цикл** — это конструкция программирования, которая позволяет выполнять один и тот же блок кода несколько раз.

Циклы применяются, когда нужно:

- повторить действие несколько раз;
- обработать элементы списка;
- выполнить вычисления до выполнения условия;
- автоматизировать повторяющиеся операции.

Основные виды циклов:

1. Цикл `for`
2. Цикл `while`

---

## Цикл for

Цикл **for** используется, когда заранее известно количество повторений.

### Общий вид

```python
for переменная in последовательность:
    действие
```

### Пример

```python
for i in range(1, 6):
    print(i)
```

Результат работы программы:

```text
1
2
3
4
5
```

Цикл `for` удобно использовать для перебора элементов списка.

```python
names = ["Анна", "Иван", "Олег"]

for name in names:
    print(name)
```

---

## Цикл while

Цикл **while** используется, когда количество повторений заранее неизвестно.

### Общий вид

```python
while условие:
    действие
```

### Пример

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

Результат:

```text
1
2
3
4
5
```

Цикл `while` выполняется до тех пор, пока условие истинно.

---

## Сравнение for и while

| Критерий | for | while |
|---|---|---|
| Количество повторений | Обычно известно заранее | Может быть неизвестно |
| Основное применение | Перебор элементов | Выполнение до условия |
| Риск бесконечного цикла | Ниже | Выше |
| Пример использования | Перебор списка | Проверка ввода пользователя |

## Примеры кода

### Сумма чисел от 1 до 10

```python
summa = 0

for i in range(1, 11):
    summa += i

print("Сумма:", summa)
```

### Вывод только чётных чисел

```python
for i in range(1, 11):
    if i % 2 == 0:
        print(i)
```

### Проверка пароля через while

```python
password = ""

while password != "12345":
    password = input("Введите пароль: ")

print("Доступ разрешён")
```
