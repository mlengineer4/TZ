# Учебный конспект: Циклы for и while

## 1. Что такое цикл

**Цикл** — это конструкция программирования, которая позволяет выполнять один и тот же блок кода несколько раз.

Циклы применяются, когда нужно:

- повторить действие несколько раз;
- обработать элементы списка;
- выполнить вычисления до выполнения условия;
- автоматизировать повторяющиеся операции.

Основные виды циклов:

1. Цикл `for`
2. Цикл `while`

---

## 2. Цикл for

Цикл **for** используется, когда заранее известно количество повторений.

### Общий вид

```python
for переменная in последовательность:
    действие
```

### Пример

```python
for i in range(1, 6):
    print(i)
```

Результат:

```text
1
2
3
4
5
```

В этом примере переменная `i` последовательно принимает значения от 1 до 5.

---

## 3. Цикл while

Цикл **while** используется, когда количество повторений заранее неизвестно.

### Общий вид

```python
while условие:
    действие
```

### Пример

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

Результат:

```text
1
2
3
4
5
```

Цикл `while` выполняется до тех пор, пока условие истинно.

---

## 4. Примеры задач

### 4.1. Сумма чисел от 1 до 10

```python
summa = 0

for i in range(1, 11):
    summa += i

print("Сумма:", summa)
```

В программе используется переменная `summa`, в которую постепенно добавляются числа от 1 до 10.

### 4.2. Вывод чётных чисел

```python
for i in range(1, 11):
    if i % 2 == 0:
        print(i)
```

Оператор `%` возвращает остаток от деления. Если `i % 2 == 0`, число является чётным.

### 4.3. Проверка пароля

```python
password = ""

while password != "12345":
    password = input("Введите пароль: ")

print("Доступ разрешён")
```

Цикл будет повторяться, пока пользователь не введёт правильный пароль.

---

## 5. Особенности циклов

- Цикл `for` удобен, когда известно количество повторений.
- Цикл `while` удобен, когда количество повторений заранее неизвестно.
- В цикле `while` важно изменять условие, иначе может получиться бесконечный цикл.
- В Python блок кода внутри цикла выделяется отступами.
- Циклы позволяют сократить количество повторяющегося кода.

## 8. Ответы на контрольные вопросы

### 1. Какие расширения имеют файлы Markdown?

Файлы Markdown обычно имеют расширение `.md` или `.markdown`.

### 2. Как создать заголовок второго уровня?

Чтобы создать заголовок второго уровня, нужно поставить два символа решётки и пробел перед текстом:

```md
## Заголовок второго уровня
```

### 3. Чем отличается маркированный список от нумерованного?

Маркированный список используется для обычного перечисления пунктов без порядка.  
Нумерованный список используется, когда важна последовательность действий.

Пример маркированного списка:

```md
- пункт
- пункт
```

Пример нумерованного списка:

```md
1. первый шаг
2. второй шаг
```

### 4. Как вставить изображение в документ?

Изображение вставляется с помощью синтаксиса:

```md
![Описание изображения](ссылка_или_путь_к_файлу)
```

### 5. Для чего нужны блоки кода и как указать язык программирования?

Блоки кода нужны для вставки фрагментов программного кода с сохранением форматирования.  
Язык программирования указывается после трёх обратных кавычек:

```md
```python
print("Hello")
```


### 6. Что произойдёт, если не поставить пробел после символа #?

Если после символа `#` не поставить пробел, Markdown может не распознать строку как заголовок. Тогда текст отобразится как обычная строка.
